# 🎮 My8Way

**My8Way** é um jogo desenvolvido em **Unity** com um objetivo simples: coletar todos os triângulos verdes enquanto evita inimigos pelo caminho.

## 🕹️ Gameplay

No jogo, você controla um player que pode se mover livremente pelo mapa. Durante a jornada, você deve:

- **Coletar todos os triângulos verdes** para completar o nível  
- **Evitar os quadrados vermelhos**, que representam inimigos  

Uma colisão com um inimigo resulta em falha, exigindo que o jogador tente novamente.

## 🎯 Controles

- ⌨️ Movimento: **teclas W, A, S, D**

## 🚀 Mecânicas principais

- Movimento em 8 direções (8-way movement)
- Sistema de coleta de pontos
- Sistema de colisão com inimigos

## 🛠️ Tecnologias utilizadas

- **Unity Engine**
- **C#**

## 📦 Como jogar

1. Inicie o jogo  
2. Use as teclas **W, A, S, D** para mover o personagem  
3. Colete todos os triângulos verdes  
4. Evite os inimigos vermelhos
